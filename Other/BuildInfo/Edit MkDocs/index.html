    Python: View local version

    MkDocs will display the site (index.md) on a local webserver 

    # activate environment Python
.venv\scripts\activate 

# activate server
mkdocs serve
    Go to http://127.0.0.1:8000/ to view Saved changes made to any file (Ctrl+S) updates and reloads the site
